Guion de Clase: Arma tu página de perfil (10 min)

0:00 - 1:30 | Introducción y Motivación
"¡Hola a todos! Soy Alejandro. En las sesiones anteriores exploramos qué son las etiquetas HTML y cómo aplicar color con CSS. Hoy vamos a integrar todo eso para construir algo genial: ¡su propia página de perfil! Es su tarjeta de presentación en la web. Trabajaremos directamente en el navegador usando CodePen, así que no necesitamos instalar nada. ¡Comencemos a programar!"

1:30 - 4:30 | Construyendo la Estructura (HTML)
"Primero, abran el enlace a CodePen que les compartí. Allí encontraremos nuestro archivo index.html. Piensen en el HTML como la estructura base de nuestra página.
Vamos a definir el contenedor principal para nuestro perfil usando la etiqueta div con la clase tarjeta-perfil. Dentro de este contenedor, añadiremos:

- Nuestra fotografía con la etiqueta img.

- Nuestro nombre usando un encabezado principal h1.

- Una breve descripción sobre nosotros con la etiqueta de párrafo p.

- Y una lista de nuestros pasatiempos favoritos usando ul para la lista y li para cada elemento.

4:30 - 7:30 | Aplicando Estilos (CSS)
"Si visualizamos el resultado ahora, se ve bastante simple, ¿cierto? Todo está alineado a la izquierda. Aquí es donde utilizamos CSS para darle estilo visual a nuestra interfaz de usuario.
Vamos al archivo style.css. Primero, definiremos un color de fondo para la página completa. Luego, daremos estilo a nuestra clase .tarjeta-perfil para que luzca como una tarjeta real: la centraremos en la pantalla, aplicaremos un fondo blanco, bordes redondeados y una sombra suave para darle profundidad."

7:30 - 9:00 | Ajustes Finales (La Fotografía)
"Es probable que nuestra foto de perfil se vea demasiado grande o con forma cuadrada. En nuestro CSS, seleccionaremos la etiqueta img, ajustaremos su tamaño y aplicaremos una regla muy útil: border-radius: 50%;. Esto transformará la imagen en un círculo perfecto. ¡Hagan la prueba en sus pantallas!"

9:00 - 10:00 | Despliegue y Cierre
"¡Miren el excelente resultado! Para el proyecto de esta sección, quiero que modifiquen los colores, las fuentes y agreguen sus propias fotografías. ¡Nos vemos en la siguiente sesión!"
