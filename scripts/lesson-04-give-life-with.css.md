## Pantalla 1: Portada 

"Hola, como estan. En esta presentación hablaré sobre cómo dar vida a una página web utilizando CSS, específicamente mediante el uso de colores y fuentes. Aunque HTML nos permite crear la estructura de una página, CSS es la herramienta que hace que esa página sea visualmente atractiva y agradable para los usuarios. A lo largo de la presentación veremos la importancia de los colores, las tipografías y algunos ejemplos prácticos."

## Pantalla 2 ¿Qué es CSS? 

"CSS significa Cascading Style Sheets o Hojas de Estilo en Cascada. Es un lenguaje que se utiliza para definir la apariencia visual de una página web.

Mientras HTML organiza el contenido, CSS permite modificar aspectos como los colores, las fuentes, los tamaños de texto y muchos otros elementos visuales.

Gracias a CSS podemos transformar una página sencilla en una interfaz más profesional y atractiva para los usuarios."

## Pantalla 3:¿Para qué ayudan los colores?
"Los colores cumplen un papel muy importante en el diseño web. No solo hacen que una página se vea más atractiva, sino que también ayudan a comunicar mensajes y emociones.

Además, permiten dirigir la atención del usuario hacia elementos importantes, como botones, títulos o avisos.

También contribuyen a la identidad visual de una empresa o proyecto, ya que muchas marcas son reconocidas por sus colores característicos.

Por estas razones, la elección de colores es una decisión importante al diseñar una página web."

## Pantalla 4: Significado de los colores

"Además de su aspecto visual, los colores suelen transmitir diferentes sensaciones.

Por ejemplo, el azul se asocia con la confianza, la seguridad y el profesionalismo. Por eso es muy utilizado por bancos, universidades y empresas tecnológicas.

El verde suele relacionarse con la naturaleza, el crecimiento y la sostenibilidad.

El rojo se utiliza para llamar la atención y transmitir energía o urgencia.

Finalmente, el naranja suele asociarse con la creatividad, el entusiasmo y la innovación.

Conocer el significado de los colores ayuda a elegir los más adecuados según el objetivo de una página web."

## Pantalla 7: Las fuentes o tipografías 

"Además de los colores, las tipografías son fundamentales en cualquier página web.

Una tipografía adecuada facilita la lectura y ayuda a transmitir la personalidad del sitio.

Algunas fuentes son más formales, otras más modernas y algunas están diseñadas específicamente para mejorar la legibilidad en pantallas.

Por eso es importante seleccionar fuentes que sean fáciles de leer y que estén alineadas con el objetivo del proyecto."

## Pantalla 8: Propiedades de las fuentes 

"CSS ofrece varias propiedades para personalizar las fuentes.

Una de las más importantes es font-family, que permite seleccionar la tipografía que utilizará el texto.

También podemos modificar el tamaño mediante font-size, el grosor mediante font-weight y el espaciado entre líneas con line-height.

Estas propiedades ayudan a mejorar la presentación del contenido y a hacer que la información sea más fácil de comprender para los usuarios."

Podemos decir que los colores aportan identidad visual y las tipografías aportan claridad y legibilidad."

## Pantalla 14: Buenas prácticas 

"Al utilizar colores y tipografías es importante seguir algunas buenas prácticas.

Primero, debemos mantener un buen contraste entre el texto y el fondo para facilitar la lectura.

También es recomendable no utilizar demasiados colores ni demasiadas fuentes en una misma página.

Lo ideal es mantener consistencia visual para que el diseño sea más profesional y agradable para los usuarios.

Finalmente, siempre debemos priorizar la legibilidad por encima de la decoración."

## Pantalla 15: Conclusión 

"Para concluir, CSS es una herramienta fundamental para mejorar la apariencia de las páginas web.

Los colores permiten transmitir emociones, reforzar la identidad visual y destacar información importante.

Las tipografías mejoran la legibilidad y ayudan a organizar el contenido.

Cuando ambos elementos se utilizan correctamente, es posible transformar una página simple en una experiencia visual mucho más atractiva para el usuario.

Muchas gracias por su atención."
