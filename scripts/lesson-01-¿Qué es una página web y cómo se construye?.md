Lección 1: ¿Qué es una página web y cómo se construye?

[0:00 – 0:30] Gancho inicial

"A ver, levanten la mano los que hoy entraron a YouTube, Instagram, TikTok o buscaron algo en Google… o sea, todos. Cada una de esas páginas que abren todos los días alguien la construyó. Y hoy les voy a mostrar cómo."

[0:30 – 1:15] ¿Qué es una página web?

"Primero lo básico: una página web es como un documento que vive en internet y que ustedes abren con un navegador, que es ese programa que usan para entrar a internet, como Chrome, Safari o Edge.

Cuando escriben una dirección o le dan clic a un link, su celular o compu le pide esa página a otra computadora que la tiene guardada. Esa página no aparece por arte de magia: está hecha con piezas de código."

[1:15 – 3:00] Las 3 piezas clave (la parte importante)

"Para que se acuerden fácil, imagínense que una página web es como una persona, y se arma con tres cosas:

La primera es el HTML, que es el esqueleto. Son los huesos de la página: el texto, los títulos, las imágenes, los botones… o sea, qué cosas hay y dónde van. Pero ojo, solo con HTML la página se ve aburrida.

La segunda es el CSS, que es la ropa y el estilo. El CSS le pone los colores, decide los tamaños, las letras bonitas, dónde se acomoda cada cosa para que se vea ordenada. Es lo que hace que una página se vea profesional.

Y la tercera es el JavaScript, que vendría a ser el cerebro y los músculos. Esto es lo que hace que la página reaccione y se mueva. Por ejemplo: cuando le dan 'me gusta' a una foto y el corazoncito se anima, eso es JavaScript.

Entonces, para resumir: HTML es la estructura, CSS es el estilo, y JavaScript es la acción. Esas tres trabajan juntas en casi toda página que ustedes usan."

[3:00 – 4:00] ¿Cómo llega la página hasta tu pantalla?

"Ahora, ¿dónde viven todos esos archivos? En una computadora especial que está prendida todo el día y conectada a internet, que se llama servidor.

Cuando ustedes entran a una página, pasa algo súper rápido: su navegador le pide los archivos al servidor, los descarga y los arma al toque para mostrarles la página completa. Y todo eso pasa en milisegundos."

[4:00 – 4:45] ¿Y cómo se construye una desde cero?

"Si yo quisiera crear una página, seguiría más o menos cuatro pasos:

Primero, la idea: para qué va a servir, por ejemplo una página de mi banda o de mi negocio. Segundo, el diseño: dibujo cómo quiero que se vea, como un boceto. Tercero, programar: ahí escribo el HTML, CSS y JavaScript. Y cuarto, la publico en internet para que todos la vean."

[4:45 – 5:00] Cierre

"Así que la próxima vez que estén scrolleando en su celular, ya saben que detrás de cada página hay HTML, CSS y JavaScript trabajando juntos. Y lo mejor de todo: con una computadora y un poco de dedicación, ustedes también pueden construir sus propias páginas. ¡Nos vemos!"
